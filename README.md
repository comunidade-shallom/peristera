# peristera

A telegram bot

## Development

```sh
task setup # install dependencies
task build # build cli
```

```sh
# build and run
task build
./bin/peristera-cli-linux-amd64
```

### Requeriments

- [Go ~> 1.18](https://go.dev/dl/)
- [Task v3](https://taskfile.dev/)
- [upx](https://upx.github.io/)
